# Self-supervised Multiscopic Vision Matching



## Usage
---
### Dataset:
```
dataset/
    TRAIN/
        SyntheData/
            000001/
                view0.png
                view1.png
                view2.png

    EVAL/

```

### Dependencies:

- Python2.7
- PyTorch (0.4.0+)
- torchvision (0.2.0+)

### Run:
```
. run.sh
```
